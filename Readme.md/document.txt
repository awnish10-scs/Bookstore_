Online Bookstore Application
Overview
This project is a backend API for an online bookstore application. It provides endpoints for managing users, books, shopping carts, and orders. Built using Spring Boot, this application covers CRUD operations and integrates with a relational database.

FeaturesUser Management: Register, retrieve, update, and delete users.
Book Management: Add, retrieve, update, and delete books.
Shopping Cart Management: Add, remove, update, and retrieve items in the shopping cart.
Order Management: Place, retrieve, and cancel orders.
Prerequisites
Java JDK 17 or later
Maven
A relational database (e.g., MySQL, PostgreSQL)
IDE (e.g., VS Code, IntelliJ IDEA)
